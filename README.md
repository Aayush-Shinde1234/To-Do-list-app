# Task Manager

A simple Python command-line application to manage tasks. Users can add, remove, and view tasks, with all tasks stored in a text file.

## Features

* Add multiple tasks
* Remove existing tasks
* View all saved tasks
* Store tasks in a `tasks.txt` file
* Simple menu-driven interface

## Requirements

* Python 3.x

## Menu Options

1. ADD
2. REMOVE
3. SHOW
4. EXIT


## Example


1. ADD
2. REMOVE
3. SHOW
4. EXIT

Enter your choice (1:4) : 1
How many task do you want to add : 2

Enter task 1 : Learn Python
Enter task 2 : Build a Project

## Files Used

* `tasks.txt` – Stores all tasks permanently.
